Arquitetura e organização de computadores

Uma visão geral

ARQUITETURA - atributos relacionados a lógica de programação visíveis a um programador de linguagem de máquina (assembly);
Exemplos de atributos arquiteturais: conjunto de instruções (instruction set), número de bits usados para representar vários tipos de dados, mecanismos de entrada e saída, e técnicas de endereçamento de memória.
ORGANIZAÇÃO - as unidades operacionais e sua interconexão, como as instrução estão formadas; Exemplos de atributos organizacionais: detalhes de hardware transparentes ao programador, tais como sinais de controle, interface entre o computador e os periféricos, tecnologia de mémória usada, etc.

EXEMPLO: é uma questão de projeto ARQUITETURAL se o computador deve ter instrução de multiplicação, mas é uma questão ORGANIZACIONAL se a instrução deve ser implementada com uma unidade de multiplicação ou através de repetidas somas.

Os principais componentes de um computador são : PROCESSADOR (CPU), que é responsável pelo controle do sistema operacional e por processar os dados e informações. A MEMÓRIA PRINCIPAL, armazena dados e instruções, o SISTEMA DE INTERCONEXÃO, realiza a comunicação entre esses dois primeiros e o os dispositivos de ENTRADA E SAÍDA, que movimenta dados entre computador e sistema interno.

O computador e seus componentes

HARDWARE: processador, memória, placa-mãe (interconexão). SOFTWARE: básicos - SO, BIOS (primeiro programa, sistema de entrada e saída, ele reconhece os componentes de hardware instalados, verifica se todos estão funcionando e passa o comando do computador ao sistema operacional, provendo a e todos os programas já instalados qndo se compra - e aplicativos.

O computador transforma dados em informação, através do hardware (parte física) e do software (parte lógica) que são os componentes responsáveis pelo processamento destes dados, transformando-o saída de informações.

BARRAMENTOS

Conjunto de linhas (condutor elétrico ou fibra ótica) de comunicação que permitem a interligação entre dispositivos, como a CPU, memória e outros periféricos. São essas linhas que transmitem a informação.
