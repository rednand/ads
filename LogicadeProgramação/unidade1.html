Introdução à Lógica de Programação

Conceitos Iniciais

A Lógica de Programação pode ser concebida por diversas linguagens, que nada mais são que formas padronizadas de representar um raciocínio lógico para desenvolvimento de soluções por meio de construção de algoritmos.
Um algoritmo poded ser definido como um conjunto de sequencias ou passos para execução de um objetivo.
Um algoritmo deve representar um programa, detalhando os dados e instruções que serão necessários para realiza-lo. As expressões utilizadas podem ser classificadas em 3 tipos:

  1. estruturas sequencias: não há desvios, instruções são diretas
  2. estruturas de decisão: instruções obedecem algumas condições
  3. estruturas de repetição: as instruções se repetem de frente a alguma decisão

Tipos de algoritmo

Os algoritmos devem apresentar uma sequência lógica que permita três elementos: entrada de dados, processamento de dados e saída de dados.
Os tipos de algoritmos podem ser classificados em relação à forma de representação deles, utilizando os três elementos mencionados. Podemos ter algoritmos representados por linguagem natural, fluxograma e pseudocódigo.

  • Linguagem Natural - descrição narrativa das intruções
  • Fluxograma - uso de símbolos gráficos
  • Pseudocódigo - forma de representação do algoritmo próxima a usada em linguagem de programação

História da Linguagem de Programação

As linguagens de programação não possuem, historicamente, uma data exata de início, mas a evolução começou a partir da década de 30, com os computadores elétricos. Segundo Pacievitch (on-line), Konrad Zuse publicou a linguagem de programação Plankalkül, em 1948, sem ter tido muito sucesso. Antes da programação ser passada para o computador, fazia-se uso de cartões de papel perfurados, que representavam o código dos programas. Nos anos 50 surgiram as primeiras linguagens modernas, de uso generalizado. Nas décadas de 60 e 70 foram criados paradigmas de programação de forma estruturadas, as linguagens Simula, C, Prolog e Pascal. Na década de 80 surgiram C++ e Perl. Nos anos 90 surgiram Java e Javascript, PHP, Visual Basic e Object Pascal.

Comandos de Entrada e Saída

Os comandos de entrada e saída são utilizados para uma interação do programa, permitindo a entrada e a saída de dados, em operações conhecidas como leitura e escrita de dados. Nos comandos de entrada, são realizadas instruções de leitura de dados, as quais podemos representar nos algoritmos com a palavra “ler” ou “leia”. Uma sintaxe que pode ser adotada é: leia (identificador da variável); Para comandos de saída, as instruções têm o propósito de realizar uma escrita de dados, sendo representada por palavras como “escreve” ou “escreva”.
