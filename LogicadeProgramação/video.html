Algoritmo é a sequencia de passos para se chegar a um determinado objetivo. Exemplo de receita, como sempre, etc.

Fases para desenvolver o algoritmo

Primeiro definir qual é o problema. Depois ele é feito em 3 fases:

  1. Entrada de Dados
  2. Processamento de Dados
  3. Saída de Dados
Formas de representar o Algoritmo
  1. Linguagem Natural
  2. Fluxograma
  3. Pseudocódigo
